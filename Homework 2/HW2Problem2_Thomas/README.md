# Problem 2: Black Friday Shopping Application

## Overview

This project simulates a Black Friday shopping system using two design patterns:

## Composite Pattern 
Organizes products and categories into a hierarchical structure.

## Observer Pattern
Notifies subscribed customers when new Black Friday deals become available.
