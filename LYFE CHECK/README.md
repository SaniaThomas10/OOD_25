# LYFE CHECK

## Overview
LYFE CHECK is a personal wellness tracking Java application built with **JavaFX**. The app allows users to manage daily habits, track moods, and maintain a journal. It includes an interactive dashboard with habit completion tracking, mood chart visualization, and a journal history section. The application implements design patterns and state management principles to enhance maintainability and user experience.

## Features
- **Home Screen**
  - Add, track, and mark daily habits as completed
  - Displays suggestions when all habits are completed
- **Mood Tracker**
  - Record daily moods with preset options (Happy, Excited, Relaxed, Stressed, Sad)
  - Displays mood history
  - Visualizes mood trends over the last 7 days using a line chart
- **Journal**
  - Write daily journal entries
  - View past entries with timestamps
- **Navigation Bar**
  - Easily switch between Home, Mood, and Journal screens
- **State Management**
  - Uses different `AppState` classes to handle habit completion states
